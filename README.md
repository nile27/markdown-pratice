# 표(table)

position 속성

| 값       |       의미        | 기본값 |
| -------- | :---------------: | ------ |
| static   |     기준 없음     | O      |
| relative |     요소 자신     | X      |
| absolute | 위치 상 부모 요소 | X      |
| fixed    |      뷰포트       | X      |

```
-- = 왼쪽 정렬 (default 값)
 :--: = 가운데 정렬
 --: = 오른쪽 정렬
```
